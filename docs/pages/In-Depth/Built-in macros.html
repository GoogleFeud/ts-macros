Built-in macros

ts-macros provides you with a lot of useful built-in macros which you can use inside macros. All the exported functions from this library that start with two dollar signs ($$) are built-in macros!

Important: You cannot chain built-in macros!

$$loadEnvfunction $$loadEnv$$loadEnv - Loads an env file from the provided path.
$$readFilefunction $$readFile$$readFile - Reads from the provided file and expands to the file's contents.
$$kindoffunction $$kindof$$kindof - Expands to the kind of the AST node.
$$inlineFuncfunction $$inlineFunc$$inlineFunc, $$inlinefunction $$inline$$inline - Inlines the provided arrow function.
$$definefunction $$define$$define - Creates a const variable with the provided name and initializer.
$$ifunction $$i$$i - Gives you the repetition count.
$$lengthfunction $$length$$length - Gets the length of an array / string literal.
$$identfunction $$ident$$ident - Turns a string literal into an identifier.
$$errfunction $$err$$err - Throws an error during transpilation.
$$includesfunction $$includes$$includes - Checks if an item is included in an array / string literal.
$$slicefunction $$slice$$slice - Slices an array / string literal.
$$tsfunction $$ts$$ts - Turns a string literal into code.
$$escapefunction $$escape$$escape - Places a block of code in the parent block.
$$propsOfTypefunction $$propsOfType$$propsOfType - Expands to an array with all properties of a type.
$$typeToStringfunction $$typeToString$$typeToString - Turns a type to a string literal.
$$typeAssignableTofunction $$typeAssignableTo$$typeAssignableTo - Compares two types.
$$textfunction $$text$$text - Turns an expression into a string literal.
$$decomposefunction $$decompose$$decompose - Expands to an array literal containing the nodes that make up an expression.
$$mapfunction $$map$$map - Takes a function that acts as a macro and goes over all the nodes of an expression with it, replacing each node with the expanded result of the macro function.
$$comptimefunction $$comptime$$comptime - Allows you to run code during transpilation.
$$rawfunction $$raw$$raw - Allows you to interact with the raw typescript APIs.
$$getStorefunction $$getStore$$getStore, $$setStorefunction $$setStore$$setStore - Allow you to store variables in a macro call.
